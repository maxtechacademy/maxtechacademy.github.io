Web design and development

  • Course content
Overview

Web development is the work involved in developing a Web site for the Internet or an intranet. Web development can range from developing a simple single static page of plain text to complex Web-based Internet applications, electronic businesses, and social network services. In this programme, you will be taught the nitty-gritty of website development as well as softwares and hardwares involved in buiding an interactive and accessible world class website.

Android software development

  • Course content
Overview

Android software development is the process by which applications are created for devices running the Android operating system. This programme will take you through a number of programming languages like Java and c++ and softwares employed in the development of android applications while explaining indepthly the brains behind every android application you see out there.

Graphics Design

  • Course content
Overview

Designing graphics goes beyond putting together texts and images and pasting them on a piece of paper. You will be trained on how to become a graphics designer who creates visual concepts, using computer softwares and by hand, to communicate ideas that inspire, inform, and captivate consumers and how to develop the overall layout and production design for applications such as advertisements, brochures, magazines, and reports.

Computer programming

  • Course outline
Overview

Computer coding or programming is the use of computer programming languages to give computers and machines instructions on what actions to perform. It is the way humans communicate with machines, and it allows us to create software like programs, operating systems, and mobile apps. With powerful programming languages like Python, C++ and Java at your fingertips, you can begin to create your own operating system or even a programming language more simpler than Python.

General Computer training

  • Course content
Overview

In this programme, we offer different courses ranging from computer maintenance and software installations to using some general application softwares like the Microsoft office. We will train you to become an expert at word processing, rendering presentations with microsoft powerpoint, professional data analysis with microsoft excel, design invoices, receipts for companies, industries, shopping malls, supermarkets and the likes.

Electronics Repair and servicing

  • Course content
Overview

This programme will take you through the fundamentals and building blocks of computer systems. This will aid your understanding of how computers operate to display images, graphics. You will be trained in identification and suitable as well as substitute replacement of broken electronics components such the transistor, resistor, inductor, capacitor and other components required for proper functioning of computer systems. The beauty of this lies in the fact that you can apply the knowledge acquired to other electronic devices like Television sets, radio, torches, phones and any electronic gadget you can think of.

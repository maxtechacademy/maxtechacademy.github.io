Web development is the work involved in developing a Web site for the Internet or an intranet. In this programme, you will be taught the nitty-gritty of website development as well as softwares and hardwares involved in buiding an interactive and accessible world class website.

Computer programming

Overview

Computer coding or programming is the use of computer programming languages to give computers and machines instructions on what actions to perform. It is the way humans communicate with machines, and it allows us to create software like programs, operating systems, and mobile apps. With powerful programming languages like Python, C++ and Java at your fingertips, you can begin to create your own operating system or even a programming language more simpler than Python.

Electronics Repair and servicing

Overview

This programme will take you through the fundamentals and building blocks of computer systems. This will aid your understanding of how computers operate to display images, graphics.
